# Veo3 Cinematic Prompts

Open-source collection of cinematic prompts for AI video generation.

- Viking POV
- Dream Environments
- Surreal Horror
- Futuristic News
- Atmospheric Loops

## Example

A hyper-realistic Viking standing in a storm on a wooden ship, cinematic lighting, handheld camera movement, volumetric fog, dramatic atmosphere, realistic ocean simulation, immersive sound design.

## Goals

This project explores AI-assisted storytelling, cinematic experimentation and generative media workflows.
